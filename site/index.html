The BOLD programming language

Strictly typed, explicitly written, and honest at compile time. BOLD reads like intent and refuses to guess, so the mistakes other languages find in production, BOLD finds before the program ever runs.

// bonus.bold, running today through the bld toolchain import system struct Developer { name: String experienceYears: Int isActive: Bool } fn calculateBonus(dev: Developer, rating: Float) -> Float { if dev.experienceYears > 5 { return rating * 1500.00 } return rating * 500.00 } fn main() { let senior = Developer { name: "Andras", experienceYears: 8, isActive: true } system.print(calculateBonus(senior, 4.5)) }
⚖
Strict static typing, zero implicit conversions

Int plus Float is a compile-time error with a diagnostic that names the fix. What you wrote is what runs.

⚡
The bld toolchain, built completely in Rust

One native binary, no runtime dependencies. Programs check in microseconds. bld run, bld check, bld fmt.

▢
Two layers, one language

A typed core for logic and data, and a declarative app layer that compiles straight to finished, beautiful web apps with the BOLD design system built in.

✦
Learnable in an afternoon

The full Learn BOLD course ships with the toolkit, and the playground teaches the app layer live in your browser. Never obscure, always ours.
